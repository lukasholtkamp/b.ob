Detailed Description

Internal members of class. This is the pimpl idiom, and allows more flexibility in adding parameters later without breaking ABI compatibility, for robots which link TeleopTwistJoy directly into base nodes. Implementing and declare all neccesary parameters, functions, variables, subscribers and publishers.
